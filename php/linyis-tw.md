## 💰 理財功能（lys.htm 第二個 Tab）

### 功能定位
投資組合管理系統，紀錄並計算各項投資標的之配息與收益。

### 表格欄位說明

| 欄位 | 功能 |
|-----|------|
| + | 新增投資項目 |
| ⭐ | 備份資料 |
| 利 | 利息/股息 |
| 價 | 買入價格 |
| 數量 | 股數/單位數 |
| 成本 | 總成本 (數量×單價) |
| 月 | 月配息金額 |
| 季 | 季配息金額 |
| 年 | 年化收益 |

### 類別 (n.com)

| 代號 | 類別 | 說明 |
|-----|------|------|
| 1 | 元大 | 包含股票、ETF |
| 2 | 國泰 | 包含股票、ETF |
| 3 | 中信 | 包含股票、ETF |
| 4 | 股票 | 一般股票 |
| 5 | 現金 | 現金部位 |
| 6 | 保險 | 保險保單 |
| 7 | 貸款 | 負債項目 |
| 8 | 借卷 | 融券部位 |
| 9 | 費用 | 固定支出 |

### 特殊功能

1. **配息日 badge**
   - 元大、中信、國泰、股票類別
   - 顯示配息月份與日期

2. **貸款特殊顯示**
   - 「價」欄位：每月應付款
   - 「數量」欄位：當月還本金

3. **殖利率計算**
   - 自動計算月/季/年化殖利率

4. **線上查價**
   - 可查詢個股即時股價

### 無法實現的功能

- **五大類分類**：股票、債券、現金、原物料、房地產
  - 原因：現有類別（元大、中信等）無法對應到五大類
